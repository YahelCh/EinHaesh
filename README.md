# EinHaesh
Digital Interactive Map for Fire &amp; Rescue Coordination


מיד בהפעלת התראת אש (לפיד בוער) האפליקציה תעלה אוטומטית לסוהרים בכלא. המסך מחולק לשני חלקים,
בצד ימין אזור של דיווחים ועדכונים- אביבה
ובשמאל מפה (כעין שרטוט אדרכלי של הכלא) עליה מסומנים כבר מראש נקודות ציון רלוונטיות לארוע לפיד בוער (הידרנטים/ לוחות חשמל ותקשורת/ ברז גז/ ארונות כיבוי אש)
בצד יהיה מקרא מפה
על המפה יהיו מצויינים שמות התאים ומספר האסירים במקום
על המפה יהיה ניתן להוסיף סימון בנגיעה לאחר בחירת אייקון ארגז כלים בהתאם לפעולה שבוצעה ע"י הסוהר (סגירת כרז גז/ הורדת שאלטר/ דליפת עשן)
בנוסף ניתן יהיה להוסיף דיווח בהתאם למיקום שנלחץ בטקסט או בהקלטה.- אביבה
בצד שמאל יעלו הנחיות לסוהר בהתאם למיקום שנמצא בו (כגון שנמצא בסמוך ללוח חשמל) - מנדי
אפשרות לנווט באמצעות המפה כגון ווייז בתוך המבנה.- טובה
ממיקום המשתמש הנוכחי ועד לנקודה שלוחץ עליה כיעד- טובה
תמיד יופיע בצד כפתור "קח אותי החוצה" שיציג לו דרך מילוט המהירה ביותר 


באופן זה המפה תציג את התקדמות הארוע בצורה ויזואלית כך שניתן יהיה לקלוט את המצב במבט עין
